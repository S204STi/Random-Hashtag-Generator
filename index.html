This is an early project of mine from my time in Galvanize, a webdev bootcamp. The inspiration for it was seeing friends on social media using insanely long chains of hashtags on nearly every post, and wanting to have a quick and easy way to mock them into a state of deep depression. This website was built using primarily HTML and JS, with some Bootstrap implementation. The API I'm using to generate the random text is Randomtext.me, to whom I am deeply grateful. Check out their stuff for your own projects. -S20

Have you ever wanted to mock a friend for using a ridiculously long hashtag chain on social media? Now you can easily generate random gibberish tag chains of absurd length, just for that purpose!
